发表于 2019-08-30 更新于 2019-10-03 分类于 Reserve 阅读次数：

Third…

寄存器(内存访问)

(一)

1.字单元:

由两个地址连续的内存单元组成.

2.DS和[address]

DS 通常存放访问数据的段寄存器

[…] 表示内存单元

ps:

mov bx,1000H

mov ds,bx

8086不支持直接将数据送入段寄存器

mov ds,1000H
//非法的

(二)

栈

栈的操作规则: LIFO(Last In First Out) 先进后出

入栈指令

push 寄存器/段寄存器/内存单元

出栈指令

pop 同上

ps:都是以字为单位进行

SS:SP

任意时刻,SS:SP指向栈顶元素.

push和pop执行,从其中得到栈顶地址

执行过程

push ax

SP = SP-2 SS:IP以当前栈顶前面的单元为新的栈顶

然后将ax内容送入栈中

pop ax

则相反 先出栈而后 SP = SP+2

栈顶超界(栈溢出)

编程时需要注意入栈与出栈数据
