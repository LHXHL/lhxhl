3.总线

数据总线

  1. 是CPU与内存或其他器件之间的数据传送的通道。

  2. 数据总线的宽度决定了CPU和外界的数据传送速度。

  3. 每条传输线一次只能传输1位二进制数据。eg: 8根数据线一次可传送一个8位二进制数据(即一个字节)。

  4. 数据总线是数据线数量之和。

地址总线

  1. CPU是通过地址总线来指定存储单元的。

  2. 地址总线决定了cpu所能访问的最大内存空间的大小。eg: 10根地址线能访问的最大的内存为1024位二进制数据(1B)

  3. 地址总线是地址线数量之和。

    控制总线

  4. CPU通过控制总线对外部器件进行控制。

  5. 控制总线的宽度决定了CPU对外部器件的控制能力。

  6. 控制总线是控制线数量之和。

内存的认识

CPU通过内存编号的读写,然后控制部件

cpu将所有部件看成是内存地址

ROM: 只允许读取,不允许写入 –不通电数据依然保存

RAM: 允许读取,允许写入 –不通电数据则消失
